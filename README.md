언리얼 5에서 Character를 사용하고 있다면, 가장 흔한 이유는 Ground Friction만 바꾸는 것으로는 충분하지 않기 때문이야.

다음 순서대로 확인해봐.

1. 정말 Character를 사용하고 있니?
ThirdPersonCharacter 기반이면 아래 값을 바꿔야 해.

CharacterMovement
    ↓
Set Ground Friction
이게 맞는 노드인지 확인.

2. Braking Deceleration도 같이 바꿔야 함 (가장 중요 ⭐)
Ground Friction만 0으로 만들어도

CharacterMovement
    ↓
Braking Deceleration Walking
값이 크면 금방 멈춰버려.

얼음에 들어가면

Ground Friction = 0.1
Braking Deceleration Walking = 0
얼음에서 나오면

Ground Friction = 8
Braking Deceleration Walking = 2048
처럼 원래 값으로 돌려줘.

3. Acceleration도 확인
Max Acceleration이 너무 크면 미끄러지는 느낌이 잘 안 날 수도 있어.

tps 3인칭 슈팅 방식

엄폐물을 이용해 장전하고 공격을 피하면서 적을 공격하며 웨이브를 클리어하는 방식

엄폐물은 데미지를 받으면 사라지고 일정 시간 지나면 다시 나타남

플레이어는 좌우로 이동하고 앞에 엄폐물이 있으면 엄폐할수 있고 엄폐시에 장전을 할 수 있음

플레이어는 체력 바 가 있고 체력이 다 깍이면 사망함 웨이브가 끝날때 마다 일부 회복됨

플레이어가 그냥 공격하면 탄퍼짐이 심하고 이동 속도가 그대로 이지만
 우클릭을 눌러 줌인 상태에서 공격을 하면 탄퍼짐이 적고 이동속도가 느려짐

한 웨이브에는 로봇이 적으로 등장하고 위치는 랜덤으로 등장하여 일정 수의 로봇을 처치해야 웨이브 끝남

로봇은 일반 로봇, 저격수, 방패봇 이 있고 추가 될수 있지만 일단 여기까지만 있음

저격수는 먼저 레이저로 조준 위치 정하고 일정 시간후 공격

방패봇은 방패를 들고 일정 시간에 방패를 내리고 공격을 함 방패를 내리고 조금 시간이 주어짐 이때 공격해야함

보스는 3번째 웨이브 마다 등장(일반 웨이브, 일반웨이브, 보스전, 일반웨이브, 일반웨이브, 보스전,... 이런 순서로)

보스는 일반 공격이 기관총이고 기관총 후에는 미사일을 발사함 이떄 엄폐를 해도 공격을 받음 미사일 공격 범위를 벗어나야함

# 🎯 게임 컨셉

**Cover Assault**는 엄폐물을 활용하여 적의 공격을 피하고,
적절한 타이밍에 장전과 사격을 반복하며 웨이브를 클리어하는
TPS(Third Person Shooter) 게임이다.

플레이어는 제한된 엄폐물을 효율적으로 활용해야 하며,
엄폐물은 지속적인 공격을 받으면 파괴되고 일정 시간이 지나면 다시 생성된다.

각 웨이브마다 다양한 로봇이 등장하며,
3번째 웨이브마다 보스가 등장하여 색다른 전투를 제공한다.

---

# ⭐ 게임 특징

- TPS(3인칭) 슈팅
- 엄폐 시스템
- 엄폐물 파괴 및 재생성

- 랜덤 적 스폰
- 다양한 적 AI
- 보스 패턴 전투

```

   │
```

```
엄폐물 HP 감소
   │
   ▼

   │
   ▼
엄폐물 파괴
   │
   ▼
```

플레이어는 한 곳에 오래 머무를 수 없으며,
